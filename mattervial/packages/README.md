
## Supporting Packages

To facilitate the implementation of descriptor-oriented featurizers like the latent MatMiner features ($l$-MM) and latent 
Orbital-Field Matrix features ($l$-OFM), we provide the following software packages. 
They are installed by default as dependencies with the `MatterVial` package.

***

* **[Autoencoder Tools](https://github.com/rogeriog/autoencoder_tools)**   
    A toolkit for creating autoencoders used to compress a dataset of descriptors into a latent space representation.

***

* **[MEGNet Model Tools](https://github.com/rogeriog/MEGNetModelTools)**  
    Crafted to simplify hyperparameter tuning in MEGNet models, particularly for reproducing the GNN (Graph Neural Network) featurizers that serve as a proxy for latent space features.

***


* **ROOST** 
    The core code implementation necessary to employ and train new ROOST (Representation of Stoichiometry) models.
